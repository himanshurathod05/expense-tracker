# expense-tracker
# 💰 Personal Expense Tracker

A full-stack expense management web application built using **Django** that allows users to track daily expenses, set monthly budgets, and monitor spending in real time through a modern dashboard interface.

---

## 🚀 Key Features

- Add and view categorized expenses
- Set and track monthly budget
- Automatic calculation of total spent and remaining balance
- Dynamic budget warning when limit is exceeded
- Clean dashboard layout with sidebar navigation
- Dark Mode toggle with saved user preference
- Smooth UI animations and modern styling

---

## 🛠 Tech Stack

- **Backend:** Django (Python)
- **Database:** SQLite (Django ORM)
- **Frontend:** HTML5, CSS3, JavaScript
- **Version Control:** Git & GitHub

---

## ⚙️ Run Locally

```bash
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
pip install django
python manage.py migrate
python manage.py runserver
```

Visit: `http://127.0.0.1:8000/`
